# 尚未冻结的开放问题

这些问题不阻塞 Recorder Core，也不得在 Version 1 中实现。Version 1 的媒体所有权、默认不删除、磁盘三级保护和 M6 冻结清单已经由 `V1_FREEZE_SPEC.md` 冻结；下面只保留研究期选择：

1. 超出目标机器 M6 冻结配置后的长期媒体保留时长，以及是否采用分层降码率归档
2. 最终书桌摄像头型号、机位和照明方案
3. 可穿戴设备是否保存连续视频还是低频帧加事件片段
4. 云端视觉模型的供应商和成本预算
5. 数学、英语和 408 的首批题库来源
6. 知识图谱拆分粒度
7. 主动追问采用语音、纸笔还是屏幕展示为主
8. 心理和健康提醒的具体阈值

所有问题都应在独立实验阶段解决，不得扩大 Version 1 范围。

如果某个问题开始影响 M0-M6 的可验收性，必须先提升为 `DECISIONS.md` 中的显式决策并重新审查范围，不能在实现中临时选择。
