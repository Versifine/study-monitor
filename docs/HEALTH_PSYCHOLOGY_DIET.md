# 健康、心理和饮食边界

> Version 1 边界：只允许将已有健康数据作为通用 JSON Evidence 导入；不实现健康平台连接器、心理/饮食推断或提醒阈值。

## 1. 心理状态

系统只做相对个人基线的行为偏离预警，例如：

- 睡眠时间和入睡时间变化
- 活动量下降
- 夜间手机使用增加
- 学习会话碎片化
- 长期不离开房间

不得直接输出医学诊断或单一抑郁分数。

## 2. 混杂因素

必须考虑感冒、咖啡因、药物、考试周、旅行、社交和设备漏记等替代解释。

## 3. 饮食

自动信号包括：

- 外卖和购买记录
- 进食时间
- 餐前餐后图像
- 食物类别
- 剩余量
- 可选称重餐垫

输出热量区间和不确定性，不提供虚假精确值。

## 4. 健康数据

- 步数
- 睡眠
- 心率
- 运动
- 体重
- 环境温湿度

必须区分未佩戴和真实无活动。
